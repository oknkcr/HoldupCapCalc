# HoldupCapCalc

**Holdup capacitor calculator** tool calculates the capacitance based on the required energy.

## Description

Hold-Up is when the power supply keeps providing power to the load before it drops below a certain voltage level.

## How to Use

Just open **THoldupCapCalc.html** in your browser.
